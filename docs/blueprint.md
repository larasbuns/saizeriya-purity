# **App Name**: Saizeriya Fan Finder

## Core Features:

- Quiz Generation: Dynamically generate a quiz question with a list of Saizeriya outlets and some decoys in Singapore.
- Outlet Selection: Allow users to select multiple outlets from a list.
- Quiz Submission: Submit the selected outlets for evaluation.
- Result Evaluation: Use AI to determine if the user selected all the correct Saizeriya outlets in Singapore and tailor a humorous and relevant comment.
- Comment Display: Display a pre-defined, lighthearted comment based on the quiz results.

## Style Guidelines:

- Primary color: Burgundy (#800020) reminiscent of Saizeriya's branding.
- Background color: Off-white (#F5F5DC) to provide a clean and readable surface.
- Accent color: Gold (#FFD700) to highlight important elements such as the submit button.
- Body and headline font: 'PT Sans' for a modern yet approachable feel.
- Use simple, recognizable icons for navigation and quiz elements.
- A clean and straightforward layout, easy to navigate on mobile and desktop.
- Subtle animations on quiz submission and result display for enhanced user experience.
